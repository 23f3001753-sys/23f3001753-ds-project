# IITM LLM Analysis Quiz Project  
### Student ID: 23f3001753  
### Email: 23f3001753@ds.study.iitm.ac.in

This project implements an API endpoint that:
- Accepts POST requests from the IITM quiz server
- Validates email + secret  
- Opens the quiz URL (with JavaScript support)
- Scrapes / extracts data
- Solves the quiz task using Python + ML/analysis code
- Submits the answer back to the quiz server
- Repeats until completion

Tech stack:
- FastAPI
- Uvicorn
- Playwright (Headless Browser)
- Pandas / NumPy
- Requests
- Python 3.10+
